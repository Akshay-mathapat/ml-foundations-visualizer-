# 📊 ML Foundations Visualizer

🚀 An interactive Streamlit web app to visualize core Machine Learning concepts like vector similarity, derivatives, and gradient descent.

---

## 🌟 Features

### 🔢 Vector Similarity

* Compute Dot Product
* Cosine Similarity
* Euclidean Distance
* Angle between vectors
* 2D Visualization

### 📉 Function & Derivative Visualizer

* Input custom mathematical functions
* Automatic symbolic differentiation (SymPy)
* Plot function and derivative
* Clean visualization

### 📊 Gradient Descent Simulation

* Interactive learning rate & iterations
* Step-by-step optimization visualization
* Loss vs Iterations graph
* Animated convergence

---

## 🛠️ Tech Stack

* Python 🐍

## 🚀 Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/ml-foundations-visualizer.git
cd ml-foundations-visualizer
pip install -r requirements.txt
streamlit run app.py
```
